- Implementação (Fases 6 e 7):

O programa implementa todas as características solicitadas.

- Como Executar:

Para executar a máquina virtual, é necessário baixar os arquivos .java disponibilizados e abri-los em seu IDE ou editor de texto preferido.
Após isso, execute o arquivo MySystem.java. Com isso feito, um menu de ações possíveis da thread Shell será apresentado no terminal.

- Observações 

OBS: Uma incoveniência do sistema implementado é o fato de que não existe uma maneira prática de descobrir se o input que será digitado
está sendo computado para o scanner do Shell ou do Console (threads concorrentes), sendo necessário olhar pelos últimos prints no terminal
qual dessas threads solicitou o input do usuário primeiro.
